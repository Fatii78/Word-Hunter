### ***Description***
#### This project is a word guessing game where the player chooses a category (Animals, Foods, or Jobs) and tries to guess a hidden word by selecting letters from an on-screen keyboard.
***

### ***Technologies Used***
* HTML
* CSS
*  JAVASCRIPT
***

### ***Getting Started***

##### The player chooses a category: Animals, Foods, or Jobs.

##### The player tries to guess the hidden word by selecting the correct letters.

##### When the word is fully guessed, a “Round Complete ✔” message appears to celebrate success.

##### The project was deployed and can be viewed [here](https://word_hunter.surge.sh/).
